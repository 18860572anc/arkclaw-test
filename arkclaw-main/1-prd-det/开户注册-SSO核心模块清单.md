# 开户注册、火山绑定与 SSO 开通核心模块清单

| 角色 | 功能模块 | 功能点 | 是否已有原型 |
|---|---|---|---|
| 客户 | 开户与购买 | 完成手机号注册、企业信息填写、销售归属绑定、套餐下单和付款 | 部分已有 |
| 客户 | 开通与进入 | 查看企业开通进度，完成企业级激活后进入已授权的 ArkClaw 工作台 | 部分已有 |
| 客户管理员 | 工作台概览 | 查看企业空间状态、资源用量、待办事项、开通状态和常用入口 | 已有 |
| 客户管理员 | 组织与成员 | 管理企业成员、角色权限、成员加入/移除和席位分配；成员不需要手动输入激活码 | 已有 |
| 客户管理员 | Claw 管理 | 查看、创建和管理企业 Claw，使用模板快速创建业务助手 | 已有 |
| 客户管理员 | 能力中心 | 管理技能、应用、知识库等企业可用能力，并配置给 Claw 使用 | 已有 |
| 客户管理员 | 资源与账单 | 查看企业开通状态、席位池、套餐、账单、用量和续费/扩容状态 | 部分已有 |
| 客户管理员 | 企业配置 | 配置品牌、员工端工作台、网络、安全和审计相关设置 | 已有 |
| 客户管理员 | 运维与监控 | 查看批量运维、可观测数据、审计日志和异常告警 | 已有 |
| 客户成员 | 成员工作台 | 进入员工端工作台，查看自己可用的 Claw、应用和企业通知 | 部分已有 |
| 客户成员 | 使用 Claw | 使用已授权的 Claw 完成日常业务任务，不管理企业级配置 | 已有 |
| 客户成员 | 知识与应用 | 使用管理员已授权的知识库、技能和应用能力 | 已有 |
| 客户成员 | 个人与安全 | 管理个人资料、登录安全和个人使用记录 | 部分已有 |
| 销售 | 获客与客户经营 | 通过分享链接、邀请码、公共池线索和客户池完成获客、归属、跟进和成单推动 | 部分已有 |
| 销售 | 开通与佣金跟进 | 查看客户付款、对公确认、开通任务、企业激活、席位池和佣金状态，推动客户补资料或完成后续处理 | 部分已有 |
| 官方后台 | 客户与销售管理 | 管理企业客户、销售归属、公共池分配、归属冲突和销售离职继承 | 部分已有 |
| 官方后台 | 订单与财务确认 | 管理标准套餐、定制订单、客户订单、对公审核和异常订单 | 部分已有 |
| 官方后台 | 开通任务与火山资源 | 管理 paid 后的企业开通任务，创建或绑定火山账号、空间、资源，并处理失败重试 | 部分已有 |
| 官方后台 | 企业激活与席位池 | 管理企业级激活入口、统一登录授权、企业席位池开通、火山侧席位/额度同步和开通异常 | 部分已有 |
| 官方后台 | 运营支撑 | 管理通知、审计、佣金确认、退款冲销和开通异常跟进 | 部分已有 |
| 后端系统 | 平台基础能力 | 支撑身份权限、用户企业关系、注册归属、订单支付和对公确认 | 部分已有 |
| 后端系统 | 开通集成能力 | 支撑开通任务、火山 API 对接、统一登录、企业激活入口、席位池权益和异常重试 | 暂无 |
| 后端系统 | 运营计算能力 | 支撑业务通知、监控告警、佣金计算、退款冲销和全链路审计 | 部分已有 |
| 火山 / 底层能力平台 | 底层资源能力 | 提供账号/空间/资源创建、统一登录接入、席位/额度开通、结果查询和异常回执 | 暂无 |
