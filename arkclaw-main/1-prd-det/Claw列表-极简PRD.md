# Claw 列表 - 极简 PRD

## 目标

提供一个 Claw 列表页，让用户快速看到当前所有 Claw 示例，以及每个 Claw 现在是谁在使用。

## 页面用途

- 展示当前租户下已有的全部 Claw 示例。
- 明确显示每个 Claw 的归属用户。
- 支持按员工 Claw / 共享 Claw 查看。

## 页面展示字段

- 名称 / ID
- 状态
- 归属用户
- 用户邮箱
- 席位等级
- 默认模型
- 当前版本
- 创建时间

说明：
归属用户是本页最核心信息，目标是让用户一眼知道“这个 Claw 现在是谁在用”。

## 页面操作

单条 Claw 保留 3 个操作按钮：

- 启动
- 停用
- 自动修复

说明：
这 3 个按钮当前先作为能力占位，后续需要开发人员与火山方确认是否可以提供对应接口或控制能力，再决定是否接入真实能力。

## 页面保留能力

- 搜索
- 导出当前列表
- 刷新
- 分页

## 当前不需要

- 回收站
- 批量删除
- 底部批量操作按钮
- 标签、部门、用户组、自动备份等非核心字段

## 备注

本页优先级以“看清谁在使用哪个 Claw”为第一目标，其他扩展管理能力暂不展开。
